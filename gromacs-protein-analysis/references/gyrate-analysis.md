# Gyrate 分析

## 概述

回转半径（Rg, Radius of Gyration）测量原子相对于质心的质量加权均方根距离，评估蛋白质紧致性和折叠状态。适用于监测紧致性变化、检测折叠/展开转变、比较不同状态、验证蛋白质稳定性。

## 工作流程

### 输入文件
- `-s`：结构文件（.tpr）
- `-f`：轨迹文件（.xtc/.trr）

### 命令
```bash
gmx gyrate -s md.tpr -f md.xtc -o gyrate.xvg
```

### 参数选择

**原子组选择**：
- **Backbone**：主链原子（推荐）
- **C-alpha**：Cα 原子，计算效率高
- **Protein**：所有蛋白质原子

**重要参数**：
- `-o`：输出文件（.xvg 格式，时间-Rg）
- `-nm`：输出每个轴（x, y, z）的 Rg
- `-b`：起始时间
- `-e`：结束时间

### 输出
- **gyrate.xvg**：时间-Rg 数据
  - 第 1 列：时间
  - 第 2 列：总 Rg（单位：nm）
  - 第 3-5 列：Rg_x, Rg_y, Rg_z（使用 `-nm` 参数）

### 可视化
```bash
dit xvg_show -f gyrate.xvg -x "Time (ns)" -y "Rg (nm)" -t "Radius of Gyration"
```

## 结果解释

### Rg 值范围

| Rg (nm) | 蛋白质大小 |
|---------|-----------|
| < 1.0 | 小蛋白质（< 50 残基） |
| 1.0-1.5 | 小到中等（50-100 残基） |
| 1.5-2.0 | 中等（100-200 残基） |
| 2.0-2.5 | 中到大（200-300 残基） |
| 2.5-3.0 | 大（300-500 残基） |
| > 3.0 | 非常大（> 500 残基） |

### Rg 模式
- **稳定 Rg**：围绕常数波动（~2-5%），蛋白质稳定
- **下降 Rg**：蛋白质变得更加紧密
- **上升 Rg**：蛋白质膨胀或展开
- **各向异性 Rg**：不同轴 Rg 值差异大，蛋白质细长

### 紧致性评估
- **紧密球形**：低 Rg，各向同性
- **伸展构象**：高 Rg，各向异性
- **部分展开**：中等 Rg，可能漂移
- **完全展开**：高 Rg，各向同性（无规卷曲）

## 常见问题

**Q: Rg 显示突然跳跃？**  
A: 检查PBC周期性问题，应用 周期性校正。

**Q: Rg 持续增加？**  
A: 可能蛋白质展开，检查结构或延长模拟时间。

**Q: 各轴 Rg 差异大？**  
A: 可能是天然细长蛋白质（如纤维状蛋白），属正常现象。
