# RDCM 分析

## 概述

残基距离接触矩阵（RDCM, Residue Distance Contact Matrix）计算残基对之间的平均距离，分析残基间空间关系和长程接触。适用于分析接触和距离、识别长程相互作用、研究折叠/展开转变、验证实验接触图。

## 工作流程

### 输入文件
- `-s`：结构文件（.tpr）
- `-f`：轨迹文件（.xtc/.trr）

### 命令
```bash
gmx mdmat -s md.tpr -f md.xtc -mean rdcm.xpm
```

### 参数选择

**原子组选择**：
- **Protein**：所有蛋白质原子（推荐）
- **Backbone**：主链原子
- **C-alpha**：Cα 原子

**重要参数**：
- `-mean`：计算平均距离矩阵
- `-b`：起始时间
- `-e`：结束时间

### 输出
- **rdcm.xpm**：平均距离矩阵（单位：nm）

### 可视化
```bash
dit xpm_show -f rdcm.xpm -o rdcm.png
```

## 结果解释

### 距离标度（颜色表示）

| 颜色 | 距离 (nm) | 含义 |
|------|----------|------|
| 深蓝色 | < 0.5 | 紧密接触 |
| 蓝色 | 0.5-1.0 | 短程相互作用 |
| 绿色/黄色 | 1.0-2.0 | 无直接接触 |
| 红色 | > 2.0 | 遥远 |

### 接触定义
- **< 0.45 nm**：直接原子接触
- **< 0.6 nm**：短程相互作用
- **< 1.0 nm**：中程相互作用

### 结构特征
- **对角线**：短距离（序列接近的残基）
- **非对角线模式**：长程接触表示三级结构
- **结构域接触**：非对角线块表示结构域间相互作用

### 二级结构模式
- **α-螺旋 (α-helix)**：特征对角线模式（残基 i, i+3, i+4）
- **β-折叠 (β-sheet)**：β-链对的强非对角线信号

## 常见问题

**Q: 距离矩阵显示均匀高距离？**  
A: 可能蛋白质展开、原子选择不正确或模拟问题。

**Q: 距离矩阵显示噪声？**  
A: 增加模拟时间以获得更好平均，或在窗口上应用时间平均。

**Q: 意外的距离模式？**  
A: 验证蛋白质序列和结构，检查模拟不稳定性。
