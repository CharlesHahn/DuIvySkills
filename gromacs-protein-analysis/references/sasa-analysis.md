# SASA 分析

## 概述

溶剂可及表面积（SASA, Solvent Accessible Surface Area）测量蛋白质可被溶剂分子接触的表面积，提供蛋白质-溶剂相互作用和表面性质信息。适用于分析溶剂暴露、识别疏水/亲水区域、研究结合位点可及性、监测展开/聚集。

## 工作流程

### 输入文件
- `-s`：结构文件（.tpr）
- `-f`：轨迹文件（.xtc/.trr）

### 命令
```bash
gmx sasa -s md.tpr -f md.xtc -o sasa.xvg
```

### 参数选择

**原子组选择**：
- **Protein**：所有蛋白质原子（推荐）
- **Backbone**：主链原子
- **C-alpha**：Cα 原子

**重要参数**：
- `-o`：输出文件（.xvg 格式，时间-SASA）
- `-output`：输出疏水/亲水分解（如 `sas_components.xvg`）
- `-or`：输出每个残基的 SASA
- `-surface`：指定表面组
- `-probe`：探针半径（默认 0.14 nm，水分子）

### 输出
- **sasa.xvg**：时间-SASA 数据
  - 第 1 列：时间
  - 第 2 列：总 SASA（单位：nm²）
- **sas_components.xvg**（使用 `-output`）：
  - 第 1 列：时间
  - 第 2 列：总 SASA
  - 第 3 列：疏水 SASA
  - 第 4 列：亲水 SASA

### 可视化
```bash
dit xvg_show -f sasa.xvg -x "Time (ns)" -y "SASA (nm²)"
```

## 结果解释

### SASA 值范围

| SASA (nm²) | 蛋白质大小 |
|------------|-----------|
| < 50 | 小蛋白质（< 50 残基） |
| 50-100 | 小到中等（50-100 残基） |
| 100-200 | 中等（100-200 残基） |
| 200-300 | 中到大（200-300 残基） |
| 300-500 | 大（300-500 残基） |
| > 500 | 非常大（> 500 残基） |

### SASA 模式
- **稳定 SASA**：围绕常数波动（~5-10%），蛋白质稳定
- **下降 SASA**：蛋白质变得更加被掩埋（折叠或塌陷）
- **上升 SASA**：蛋白质变得更加暴露（展开或变性）
- **突然变化**：大的构象变化或配体结合/解离

### 疏水/亲水比率
- **> 1.0**：主要为疏水（可能聚集）
- **~1.0**：平衡表面（可溶性蛋白质）
- **< 1.0**：主要为亲水（高度可溶）

### 每个残基 SASA
- **< 0.1 nm²**：被掩埋残基（蛋白质核心）
- **0.1-0.5 nm²**：部分暴露残基
- **> 0.5 nm²**：表面暴露残基

## 常见问题

**Q: SASA 持续增加？**  
A: 可能蛋白质展开，检查结构或延长模拟时间。

**Q: 疏水/亲水分解不工作？**  
A: 确保使用 GROMACS 2020 或更高版本，检查 `-surface` 参数。

**Q: SASA 值异常低？**  
A: 检查原子选择，蛋白质可能过度紧密或塌陷。
